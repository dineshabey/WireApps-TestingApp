# WireApps-Testing

 =========== Getting Started en local enviroment ============

1.Clone the repository.
2.Install Composer dependencies: composer install.
3.env and configure your environment variables.
4.Run database migrations: php artisan migrate.
5.Start the development server: php artisan serve.
Usage
6.Visit the homepage at http://localhost:8000.
7.Explore the application and its features.
8.Use API collection json file for test the application
